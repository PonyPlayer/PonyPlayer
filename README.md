# PonyPlayer



## 1 技术栈

UI框架采用 QML（Qt 6.2.4）

视频编解码采用 FFmpeg

使用CMake+Ninja组织和构建项目

同时支持Windows，macOS和Linux

其中Windows打包成`msi`，macOS打包成`dmg`，`Linux`打包成`AppImage`；
