# HowToCode

## 一般标准

1. 我们使用C++20标准，请写Modern C++风格的代码；
2. 类名大驼峰，属性/字段/函数/方法名小驼峰，文件名全小写；
3. 注意clang-tidy的提示，代码要做到0-warning；
4. 公开的接口要写注释，如

```cpp
/**
 * 测试用函数
 * @param arg1 参数1 
 * @param arg2 参数2
 * @return 返回值说明
 */
int foo(int arg1, int arg2) {
    return 0;
}
```
